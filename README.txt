Aspectos Importantes del Proyecto y su correcta ejecución:

1. El proyecto fue realizado específicamente en la Herramienta NetBeans IDE 8.0.1.

2. Se utilizó una gramática de SQL que describiera el lenguaje que debiera de aceptar el proyecto. Se utilizó ANTLR
para reconocer el lenguaje, leerlo, procesarlo, etc. Se utilizó la versión 4.4 complete, la cual puede conseguir
en el siguiente enlace: http://www.antlr.org/download/index.html.

3. Para manejar las tablas, se utilizaron JSON para almacenar la información de las mismas, así como también de las bases de datos.
Esta herramienta se implementó como una herramienta adicional a NetBeans, la cual puede descargar en el siguiente enlace:
https://www.dropbox.com/s/08f9def08gocj02/json-simple-1.1.1.jar?dl=0. Cabe destacar que la versión de esta herramienta es la 1.1.1 simple.

4. El código fuente se puede accesar en el siguiente enlace:
https://github.com/JJ-Fong/DBProject1

5. Se debe descargar el código fuente del enlace anterior, así como todas las herramientas antes mencionadas y agregar las mismas
como librerías del proyecto para poder ser ejecutado correctamente.
